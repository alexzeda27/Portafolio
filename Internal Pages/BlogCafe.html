Blog De Café

Un blog dedicado a la preparación de café.

Blog de Café es un proyecto que ofrece cursos y consejos acerca de la preparación de esta bebida, ademas de contener recetas que acompañara esta rica bebida con una comida especial para la satisfacción del cliente que llegase a probar alguna de estas recetas. La parte de desarrollo de Font-End del sitio web se desarrollo con tecnólogias de maquetación como HTML5 y CSS3 usando CSS grid y responsivo apto para dispositivos de 420 a 560px.
