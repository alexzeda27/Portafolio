Bienes Raices

Un proyecto sobre compra o renta de hogares.

Bienes raices es un proyecto que incio como la compra de casas u apartamentos, pero a medida que el proyecto iba creciendo notamos que la gente también busca lugares donde puedan alojarse durante un tiempo determinado, es por ello que la renta de estos establecimientos se convirtio en una necesidad para nosotros. Mi participación en este proyecto se vio reflejada en el Front-End del sitio. Bienes raices fué creado con las herramientas de diseño web tales como HTML5 y CSS3, no se implemento JavaScript ya que no eran necesarias para este proyecto.
