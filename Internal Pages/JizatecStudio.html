Jizatec Studio

Empresa impartidora de talleres y seminarios.

Jizatec Studio es una empresa que se dedica a impartir cursos, talleres, seminarios sobre tecnologías de sistemas donde le dan al oportunidad a los estudiantes de Unitec Campus Toluca de asistir a los eventos para conocer mas sobre estas tecnologías. Este proyecto fué elaborado con HTML5, CSS3, JavaScript para la parte del Front-End y se usó PHP, Ajax, y MYSQL para la parte de Back-End y la creación de la base de datos.
